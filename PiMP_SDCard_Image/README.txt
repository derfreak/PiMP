Requirements:
RaspberryPI (Mod B rev.2 recommended)
8GB SD-Card (Class 10 or better recommended)

normal User:
Username: interface
Password: interface

Superuser:
Username: root
Password: root


Additional Information:
You need to bridge 3.3V (Pin 1) and GPIO 27 (Pin 13) of the P1 header. Otherwise, the Raspberry will automatically shut down.
GPIO 27 exported as digital input and needed to shut down the OS, if ignition switch turned off.
